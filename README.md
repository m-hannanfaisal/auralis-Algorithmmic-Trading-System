Auralis is a research-grade algorithmic trading system built as a dual-domain project — applying core Data Structures & Algorithms for efficient real-time data management, and Machine Learning for intelligent trade signal filtering.

The system ingests historical candlestick data, generates trading signals using trend and volatility indicators, filters them through an XGBoost classifier, and backtests the result against four rule-based baselines.

---

## 🏆 Key Results

| Strategy | Net R-Multiple | Notes |
|---|---|---|
| Baseline A | +15.58R | Simple breakout |
| Baseline B | +22.31R | Breakout + Asia filter |
| Baseline C | +18.44R | Retracement entry |
| Baseline D | +21.07R | Combined rules |
| **Proposed (XGBoost)** | **+94.92R** | **ML-filtered signals** |

> **R-Multiple** = return normalized by initial risk per trade. Higher = better risk-adjusted performance.

## 🧠 Dual Domain: DSA + AI/ML

### 📐 Data Structures & Algorithms

Custom-built structures designed for real-time trading data:

| Module | Structure Used | Purpose |
|---|---|---|
| `dsa/candle_store.py` | Dynamic Array + Binary Search | Fast O(log n) candle retrieval |
| `dsa/session_manager.py` | Hash Map + AVL Tree | Session-based time window management |
| `dsa/trade_manager.py` | Min-Heap | Priority-based trade risk tracking |
| `benchmark_dsa.py` | — | Empirical complexity benchmarking |

Benchmarks verify theoretical complexities (O(log n) inserts, O(1) lookups) against real data sizes up to 100,000 candles.

### 🤖 Machine Learning (XGBoost)

| Component | Detail |
|---|---|
| Model | XGBoost Binary Classifier |
| Features | Trend indicators, volatility ratios, session context |
| Labels | Profitable trade = 1, Losing trade = 0 (generated by `pipelines/labeler.py`) |
| Filter | Signals with confidence ≥ threshold passed to backtester |
| Threshold Sweep | Tested at 5%, 30%, 40%, 50%, 60%, 70%, 80%, 95% |

## ⚙️ Installation & Setup

```bash
# Clone the repo
git clone https://github.com/YOUR_USERNAME/auralis-dsa.git
cd auralis-dsa

# Install dependencies
pip install -r requirement.txt
```

---

## 🚀 Running the Project

```bash
# 1. Run DSA benchmarks & demonstrations
python auralis_dsa.py
python benchmark_dsa.py

# 2. Build signal dataset & labels
python pipelines/build_signals_dataset.py
python pipelines/labeler.py

# 3. Train the XGBoost model
python models/train_xgb_filter.py

# 4. Run full experiments (all baselines + proposed)
python run_experiment.py
python run_experiments_v2.py

# 5. Evaluate & generate plots
python evaluate_results_v2.py
python generate_dsa_figures.py
```
